## 什么是 Hermes Agent？

Hermes Agent 是一个专注于消息路由、Agent 间通信与传递保障的专职 Agent，服务于多智能体系统内部的协调与调度。

## 设计目标

- Agent 间的低延迟消息传递
- 带故障处理的可靠投递保障
- 与传输协议解耦的抽象层
