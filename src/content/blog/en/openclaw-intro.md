## What is Openclaw?

Openclaw is a framework for building autonomous multi-agent systems. It provides the primitives for defining specialized agents, wiring up inter-agent communication, and running end-to-end pipelines with clear responsibility boundaries.

## Core concepts

- Agent definitions and role assignment
- Inter-agent communication protocols
- Pipeline composition and state handoff
- Failure isolation between agents
